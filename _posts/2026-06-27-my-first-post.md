# My First post

The text surrounded by `---` is the Front Matter. Jekyll will use it to regonize metadata.

  Front Matter 常用字段

  ┌───────────────────┬───────────────────────┐
  │       字段        │         说明          │
  ├───────────────────┼───────────────────────┤
  │ layout            │ 用哪个布局文件        │
  ├───────────────────┼───────────────────────┤
  │ title             │ 文章标题              │
  ├───────────────────┼───────────────────────┤
  │ date              │ 发布时间              │
  ├───────────────────┼───────────────────────┤
  │ categories / tags │ 分类/标签             │
  ├───────────────────┼───────────────────────┤
  │ permalink         │ 自定义 URL,如 /hello  │
  ├───────────────────┼───────────────────────┤
  │ published         │ false 表示草稿,不构建 │
  └───────────────────┴───────────────────────┘
